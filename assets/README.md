# Screenshots

This folder contains screenshots for the README.md.

## Required Screenshots

1. `screenshot-main.png` - Main interface (index.html)
2. `screenshot-showcase.png` - Video showcase page (video-showcase.html)

## How to Capture Screenshots

1. Start the local server:
   ```bash
   ./start_server.sh
   ```

2. Open Chrome and navigate to:
   - http://localhost:8000/ (for main interface)
   - http://localhost:8000/video-showcase.html (for showcase)

3. Use one of these methods to capture:
   - macOS: `Cmd + Shift + 4` then drag to select area
   - Chrome DevTools: `Cmd + Shift + P` > "Capture screenshot"
   - Or right-click > "Take Screenshot" (in some browsers)

4. Save screenshots to this folder with the names above.
